UMNAAPP Maps — use from another project

1. Iframe embed

<iframe src="HOST/embedded-map?lat=12.97&lng=77.59&zoom=12"
        style="width:100%;height:420px;border:0"></iframe>

2. JavaScript SDK

<script src="HOST/sdk.js"></script>
const map = UmnaMaps.embed('#sdkMap', { center:{lat:12.97,lng:77.59}, zoom:12 });
map.on('placeClick', p => console.log('clicked', p));

3. Public REST API (data only)

Type a query and press Search.
fetch('HOST/api/public/search?q=restaurant').then(r => r.json())
